Word vector space and topic vector space

word vector space

A strict definition of word vector space model: given a text set D={d1,d2,…,dn}, and a word set W={w1,w2,…,wm}. We can get a matrix X, its size is m*n, xij means the times or the weight of word wi occurs in the text dj.

We often presents the weight as TF-IDF(term frequency-inverse document frequency) :
$$
TF-IDF_{ij}=\frac{tf_{ij}}{tf_{\bullet j}}log\frac{df}{df_i},i=1,2,…,m;j=1,2,…,n.
$$
where tfij means the frequency that wi occurs in dj, tf.j means the sum of the frequency that all terms occurs in document dj, df is the number of all documents of set D, dfi means the documents that contains term wi.

The inner multiply or cosine of two term vectors is the similarity of two documents :
$$
x_i\bullet x_j,\frac{x_i\bullet x_j}{||x_i||\ ||x_j||}
$$
The downsides about word vector space model are the polysemy and the synonymy of natural language.

topic vector space

Given a document set D={d1,d2,…,dn} and a corresponding word set W={w1,w2,…,w3}. We can get a term-document matrix matrix X. Suppose all documents have k topics, each topic is presented as a m-dimension vector on set W :
$$
t_l=[t_{1l},t_{2l},…,t_{ml}]^T,l=1,2,…,k
$$
where til is the weight of wi in topic tl, k topic vectors t1,t2,…,tk consist of a topic vector space, its dimension is k. We can get a word-topic matrix, notates :
$$
T=[t_1,t_2,…,t_k]
$$
The document dj in set D, is a vector xj in word vector space, project it into topic vector space T, we can get a vector yj :
$$
y_j=[y_{1j},y_{2j},…,y_{kj}]^T,j=1,2,…,n
$$
where ylj is the weight of document dj in topic tl. We can get a topic-document matrix Y :
$$
Y=[y_1,y_2,…,y_n]
$$

Linear transform from word vector space to topic vector space

Now we can present xj approximately using yj :
$$
x_j\approx y_{1j}t_1+y_{2j}t_2+…+y_{kj}t_k
\X\approx TY
$$

LSA algorithm

Given the number of topics k, word-document matrix X, we use truncated matrix :
$$
X \approx U_k\sum\ _kV_k^T
$$
We use uk as topic vector space, and (∑kVkT) as the presentation of document in topic vector space.

Non-negative matrix factorization

Given a non-negative matrix X≥0, find two matrixes W≥0 and H≥0, we have :
$$
X\approx WH
$$
If X is mn, W is m\k, H is k*n, k<min(m,n), NMF is a compressing method for former data.W is a topic vector space, H is document in topic vector space

NMF can be presented as an optimal question, we definite the loss function or the cost function first :

(1)square loss, non-negative matrix A=[aij]mn, B=[bij]m\n :
$$
||A-B||^2=\sum_{i,j}(a_{ij}-b_{ij})^2
$$
its lower boundary is 0 when A = B.

(2)divergence, non-negative matrix A=[aij]mn, B=[bij]m\n :
$$
D(A||B)=\sum_{i,j}(a_{ij}log\frac{a_{ij}}{b_{ij}}-a_{ij}+b_{ij})
$$
its lower boundary is also 0 when A = B.D(A||B)≠D(B||A).Then we get the optimal problem as follows :
$$
\mathop{min}\limits_{W,H}||X-WH||^2\s.t.\ W,H≥0
$$
or
$$
\mathop{min}\limits_{W,H}D(X||WH)\s.t.\ W,H≥0
$$
For square loss function, we have update rule:
$$
H_{lj}<=H_{lj}\frac{(W^TX){lj}}{(W^TWH){lj}}\W_{il}<=W_{il}\frac{(XH^T){il}}{(WHH^T){il}}
$$
for divergence loss function, we have :
$$
H_{lj}<=H_{lj}\frac{\sum_i[W_{il}X_{ij}/(WH){ij}]}{\sum_iW_{il}}\W{il}<=W_{il}{\frac{\sum_j[H_{lj}X_{ij}/(WH)_{ij}]}{\sum_jH_{lj}}}
$$
