Probabilistic Latent Semantic Analysis

2021-02-25 technology PV:

probabilistic latent semantic analysis, also called probabilistic latent semantic indexing is a kind of unsupervised learning method of topic analysis to document set by using probabilistic generated model.

PLSA model

basic idea

Given a document set, each document contains some topics and each topic is represented by some words. The topics of the document cannot be observed directly, we can use PLSA to get topics of each document. Translate document set to word-document matrix so we can get topics by using PLSA model.

generated model

Word set W={w1,w2,…,wM}, document set D = {d1,d2,…,dN}, topic set Z = {z1,z2,…,zK}, we have :

1.get N documents randomly from document set D with P(d);

2.get L topics from topic set Z with P(z|d);

3.get words from word set W with P(w|z);

we get a tripple tuple (w,z,d) set, however z is a latent variable, notate document-word co-occurence data as T, we present the result as (w,d) set, we get two generated formulas :
$$
P(T)=\prod_{(w,d)} P(w,d)^{n(w,d)}\P(w,d)=P(d)\sum_{L}P(z|d)P(w|z)
$$

Generated model suppose that w and z are conditional independent with given d :
$$
P(w,z|d)=P(w|d)P(z|d)
$$
the N and M in this directed graph should be L and N using our formulas.

co-occurence model

we can get another formula from P(T) :
$$
P(w,d)=\sum_{z\in Z}P(z)P(d|z)P(w|z)
$$
Co-occurence model suppose that w and d are conditional independent with given z :
$$
P(w,d|z)=P(w|z)P(d|z)
$$

the relationship between PLSA and LSA

PLSA model can be described under LSA model, use SVD to decompose the word-document matrix, we get X = U∑VT, PLSA model can be presented in the same form :
$$
X’=U’\sum’V’^T\X’=P(w,d){M*N}\U’=P(w|z){MK}
\\sum’=P(z)_{KK}\V’=P(d|z)_{N*K}
$$
U’ and V’ in PLSA model are non-negative and standardized, they express conditional probabilistic ditributions, however in LSA model U and V are orthogonal and don’t express probabilistic distribution.

The algorithm of PLSA

Input : word set W={w1,w2,…,wM}, document set D = {d1,d2,…,dN}, topic set Z = {z1,z2,…,zK}, co-occurence data {n(wi,dj)},i=1,2,…,M,j=1,2,…,N;

Output : P(wi|zk) and P(zk|dj)

step 1 : initialize P(wi|zk) and P(zk|dj);

step 2 : E:
$$
P(z_k|w_i,d_j)=\frac{P(w_i|z_k)P(z_k|d_j)}{\sum_{k=1}^K P(w_i|z_k)P(z_k|d_j)}
$$
M:
$$
P(w_i|z_k)=\frac{\sum_{j=1}^N n(w_i,d_j)P(z_k|w_i,d_j)}{\sum_{m=1}^M\sum_{j=1}^N n(w_m,d_j)P(z_k|w_m,d_j)}\P(z_k|d_j)=\frac{\sum_{i=1}^M n(w_i,d_j)P(z_k|w_i,d_j)}{n(d_j)}
$$
