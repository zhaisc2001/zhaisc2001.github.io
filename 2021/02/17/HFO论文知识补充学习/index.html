HFO论文知识补充学习

2021-02-17 technology PV:

1.equalization filter : In Human and automated detection of high-frequency oscillations in clinical intracranial EEG recordings, equalization filter via first-order backward differencing is proposed in signal preprocessing section, that’s not the section I wanna be involved in, so I decide not to learn about it.

2.KS-test : The Kolmogorov-Smirnov test (KS-test) tries to determine if two datasets or distribution differ significantly. It was used to indicate line length values are not normally distributed.

3.Cohen’s kappa coefficient : Cohen’s kappa coefficient (κ) is a statistic that is used to measure inter-rater reliability (and also intra-rater reliability) for qualitative (categorical) items. It was used to evaluate human reviewer reproducibility at identifying HFOs.

4.Pearson correlation coeffcient : is defined in statistics as the measurement of the strength of the relationship between two variables and their association with each other. It was used to compare the relative detection rate between detectors.

5.Wavelet entropy or Stockwell entropy : After wavelet transformation or Stockwell transformation, use Shannon entropy formula to calculate.

6.Hilbert transform : In mathematics and in signal processing, the Hilbert transform is a specific linear operator that takes a function, u(t) of a real variable and produces another function of a real variable H(u)(t). This linear operator is given by convolution with the function 1/(πt). (see § Definition) The Hilbert transform has a particularly simple representation in the frequency domain: It imparts a phase shift of ±90° (⁄₂ radians) to every frequency component of a function, the sign of the shift depending on the sign of the frequency.

7.Wilcoxon rank sum test : It is a nonparametric statistical test that compares two paired groups. The tests essentially calculate the difference between sets of pairs and analyzes these differences to establish if they are statistically significantly different from one another.

8.Mean-shift clustering : refer to https://www.geeksforgeeks.org/ml-mean-shift-clustering/

Assigns the data points to the clusters iteratively by shifting points towards the mode (mode is the highest density of data points in the region, in the context of the Meanshift).Unlike the popular K-Means cluster algorithm, mean-shift does not require specifying the number of clusters in advance. The number of clusters is determined by the algorithm with respect to the data.The downside to Mean Shift is that it is computationally expensive O(n²).

It builds upon the concept of kernel density estimation(KDE), KDE is a method to estimate the underlying distribution, it’s also called the probability density function for a set of data. Suppose we are given a data set {ui} of points in d-dimensional space, sampled from some larger population, and that we have chosen a kernel K having bandwidth parameter h. Together, these data and kernel function returns the following kernel density estimator for the full population’s density function.
$$
f_k(u)=\frac{1}{nh^d}\sum^n_{i=1}K(\frac{u-u_i}{h})
$$
The kernel function k is required to ensure that our estimate is normalized and it is associated with the symmetry of our space, hence it satisfies:
$$
1.\int K(u)du=1\2.K(u)=K(|u|)\ for\ all\ values\ of\ u.
$$
Two popular kernel functions are flat kernel function and Gaussian kernel function.

9.Spectral centroid : This feature measures the frequency corresponding to the center of the spectrum, which is expressed as
$$
f_{c}=\frac{\sum^{L/2}{k=1} \frac{k}{LT}|M(k)|^2}{\sum^{L/2}{k=1}|M(k|^2}
$$
where M(k) is the multi-window power spectral density estimate:
$$
M(k)=\sum_{l=0}^{L-1}\omega(l)x(l)e^{-j(2\pi/L)lk}
$$
with an L-point discrete fourier transform. Here, ω(l) is the Hamming window of length L.

10.Expectation-Maximization Gaussian-Mixture-Model Algorithm : https://machinelearningmastery.com/expectation-maximization-em-algorithm/#:~:text=The%20Expectation-Maximization%20Algorithm%2C%20or%20EM%20algorithm%20for%20short%2C,latent%20variable%20models%20is%20the%20expectation-maximization%20%28EM%29%20algorithm.

To solve density estimation problem with latent variables by maximum likelihood estimation, we use the expectation-maximization algorithm. It is an iterative approach cycles between two modes : E-step and M-step.

  • E-Step. Estimate the missing variables in the dataset.
  • M-Step. Maximize the parameters of the model in the presence of the data.

A mixture model is a model comprised of an unspecified combination of multiple probability distribution functions.The Gaussian Mixture Model, or GMM for short, is a mixture model that uses a combination of Gaussian (Normal) probability distributions and requires the estimation of the mean and standard deviation parameters for each.

In the EM algorithm, the estimation-step would estimate a value for the process latent variable for each data point, and the maximization step would optimize the parameters of the probability distributions in an attempt to best capture the density of the data. The process is repeated until a good set of latent values and a maximum likelihood is achieved that fits the data.

The algorithm itself is omitted here, you can read 李航. 统计学习方法. 清华大学出版社, 2012. for furthre study.

11.Singular Value Decomposition : https://ocw.mit.edu/courses/mathematics/18-06sc-linear-algebra-fall-2011/positive-definite-matrices-and-applications/singular-value-decomposition/MIT18_06SCF11_Ses3.5sum.pdf

The singular value decomposition of a matrix is usually referred to as the SVD. This is the final and best factorization of a matrix:
$$
A=U\sum V^T
$$
where U and V are orthogonal, and ∑ is a rectangular diagonal matrix. Here, A can be any matrix.

See compact SVD and truncated SVD in 李航. 统计学习方法. 清华大学出版社, 2012..
