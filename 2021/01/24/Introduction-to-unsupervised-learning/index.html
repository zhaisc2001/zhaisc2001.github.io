Introduction to unsupervised learning

2021-01-24 technology PV:

Fundamentals of unsupervised learning

Unsupervised learning is a machine learning method, which can learn the statistical patterns or internal structure from the data without labels, mainly includes clustering, dimensionality reduction and probability estimation, it can be used as pre-process of data analysis or supervised learning.

Suppose we have a training set which can be presented as a M*N matrix, each single row means a feature and each single line means a sample. The basic idea of unsupervised learning is to give some kind of “compression” to the matrix in case that we can find a potential structrue of the data. Unsupervised learning can find the horizontal structrue of data such as clustering, can find the vertical structrue of data such as dimensionality reduction, and can find both horizontal structrue and vertical structrue of data such as learning the generated model from the data.

Basic questions

Clustering

There are two kinds of clustering, one is hard clustering, the other one is soft clustering. The former cluster one sample to one class, the later cluster one sample to multiple calsses. Suppose input zone X ⊆ Rd, output zone is class set Z = {1,2,…,k}, then the clustering model is z= gθ(x) or Pθ(z|x), x∈X is the sample vector,z∈Z is the class of sample, θ is parameter, z= gθ(x) is the modle of hard clustering and Pθ(z|x) is the model of soft clustering.

Clustering can help find the potential horizontal structrue of data, as an exception, co-clustering is an algorithm of clustering, it clusters both samples and features, it learns the vertical and horizontal structrue of data in the meanwhile.

Dimensionality reduction

Dimensionality reduction is convert the sample from high-dimen zone to low-dimen zone, or convert the sample into same dimensionality zone but a proper strctrue, it will present the relationship between samples better. There are two kinds of dimensinality reduction: linear and non-linear. The model of dimensionality reduction is zi=gθ(xi),i=1,2,…,N. Dimensionality reduction can help find the potential horizontal structrue of data.

Probability model estimation

Suppose that the training data was generated by a probability model, we can learn the structrue and parameters from training data. The structrue type of the probability model was given by us, and probability model includes hybrid model, probabilistic graphic model and so on. Probabilistic model includes directed graphic model and undirected graphic model.

Probability model can be presented as Pθ(x|z), z means the number of components in hybrid model, and means the structrue of graph in probability graphic model.Soft clustering can be considered as a probability model estimation problem as well, according to the bayesian formula:
$$
P(z|x)=\frac{P(x|z)P(z)}{P(x)}\propto P(x|z)P(z)
$$
Assume that P(z) follows a uniform distribution, we just need to estimate Pθ(x|z).

Probability model estimation is used in topic analytics and graph analytics.
