Chap 1:The role of Algorithms in Computing

Algorithms

  • algorithm: A sequence of computational steps that transform the input into the output.
  • data structrue: A way to store and organize data in order to facilitate access and modifications.
  • technique: How to analysis the efficiency of an algorithm, and how to design an algorithm
  • hard problems: There are some problems no efficient solution is known, a famous subset of these problems are NP-complete.

NP-complete problems are fascinating, by the way, as the first problem proved to be NP-complete, SAT, its known upper bound has been improved to O*(1.2226m) by my schoolmates, if that’s true, it’s a really solid work An Improved Upper Bound for SAT.

Algorithms as a technology

The efficiency of a program depends more on algorithm than hardware and software, it’s a characteristic that separates the truly skilled programmers from the novices.
