3.linear neural networks

linear regression

  • assume y can be presented as weighted sum of x and noise, besides, noise is well-behaved(following a Gaussian distribution)
  • weights and bias: affine transformation of inputs
  • loss function: squared error
  • linear regression has analytic solution(how to get it?)
  • MSGD: sampling a random minibatch of examples every time we need to update, remember that we need an average gradient
  • hyperparameter : adjust them on results accessed on a separate validation dataset
  • we don’t need to find the real minimize of loss function because of generalization
  • the different meaning of inference in deep learning and statistics
  • maximum likelihood estimators: write it as log function
  • from linear regression to deep networks
