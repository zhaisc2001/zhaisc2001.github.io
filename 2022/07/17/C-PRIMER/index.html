C++ primer

1.Begining

C++至少要包含一个名字为main的函数

cin,cout,cerror,clog

  • cin.getline(char *buffer, int N);cin.get(char& var);cin.ignore();

    • cin.ignore(numeric_limits::max(),’\n’):跳过流上限的输入，直到换行符

  • cout.write(char *str, int n);cout.put(char &ch);cout.precision(int n)

    • cout << “Name : “ << name << endl
      << “Age : “ << age << endl;

  • cerr: is used when we need to display the error message immediately and does not store the error message to display later

flow of control

  • ++a,a++?
  • val in for header is local
  • reading an unknown number of inputs

class

  • #include “Sales_item.h”
  • member functions = methods

2.Variable and basic types

char is unsigned on some machines and signed on others.

  • char:’’,string:””

assign an out-of-range value to an object of signed type, result is undefined.

list initialization

Variable names:lower case;classes we define begins with an uppercase letter;multiple words should be distinguished.

cannot define a reference to a reference

void* pointers

reference is not a object

  • a const type must be explicitly initialized.

constant expression depends on types and initializers.

auto ordinarily ignores top-level const.

3.string,vector,array

Don’t use for loop if the size of vector changes in the iteration.

in vector, subscripting doesn’t add elements.

in mac os, use control+d to end the cin input.

iterator: denotes element or the position one past the last element of the container.

  • dereference the iterators to get the value.
  • loops that use iterators should not add elements to the container.

array: dimension must be a const expression

  • initialize character arrays from string literal remember that string ends with a null character.

  • No assignment or copy for arrays

  • read array from inside out to better understand the array type.

  • most places when we use arrays, the compiler automatically converts it to a pointer to its first element.array=&array[0]

    • this conversion doesn’t happen when we use decltype().

  • array subscript index can be negative.

    • int *p[4] int(*p)[4]
    • typedef int int_array[4]

4.expressions

bool values shouldn’t be used for computation

  • i<j<k:compares k to the bool result of i<j, it means it’s always true when k > 1

assignment has low precedence

The difference between i++ and ++i

  • *i++ means *(i++)

sizeof a dereferenced pointer need not to be a valid pointer

array doesn’t convert to pointer when used with decltype(),sizeof(),&,typeid

5.statement

switch(){case A: break;default:break;}

do while loop doesn’t allow declaration in condition

if-throw;try-catch

6.functions

call operator()

a function can’t return an array type or a function type.

local static object: they are destroyed when program terminates.

references and pointers arguments can change the values of the objects through function.

  • use reference to const when possible

ways to make the function know the size of array

  • end marker
  • standard library conventions
  • explicitly pass a size parameter

initializer_list

  • the elements are const values

Don’t return a reference or a pointer to the local object

The main function may not call itself

int (*func(int i))[10]

  • auto func(int i)->int(*)[10]

    • decltype(array) *func(int i)

overloaded function: same name but different parameters.

  • the main function may not be overloaded

local declaration hides outer declaration

assert: do nothing if the condition is true.

pointers point to function need parameters and return types

  • we can assign and call the function through pointers without dereferences
  • the function in return type can’t convert to pointer automatically

7.class types

const follows the parameter list means this is a pointer to const

constructor

  • can’t be declared

the only difference between class and struct is the default access level.

a class can allow another class or function to access its nonpublic members by make that class or function a friend

a mutable data number is never const

a friend declaration affects access but is not a declaration

must use a constructor initializer list to provide values for members that are const, reference, or of a class type that doesn’t have a default constructor.

a constructor that supplies default arguments for all its parameters also define the default constructor

the difference between Sales_item obj() and Sales_item obj2

member definition has access to the private member of the class

8.the IO library

io classes which are designed to support languages that use w_char, their names begin with w

no copy or assign io classes, no parameter or return type

Buffers are not flushed if the program crashes

the only way to preserve the existing data in a file opened by ostream is to specify app or in explicitly

9.sequential containers

vector

  • flexible size, fast random access, add or insert elements in the middle takes time

deque

  • double-ended queue, fast random access, fast delete/insert at front or back

list

  • double linked list, only bidirectional access, fast delete/insert any point in the list

forward_list

  • singly linked list, only sequential access in one direction, fast delete/insert any point in the list
  • doesn’t have size operation

array

  • fixed-size, faa, can’t delete or insert element
  • can’t use array a(b,e) to initialize arrays
  • the size is part of array’s type

string

  • similar vector that contains character, faa, fast delete/insert at the back

c.begin() can get a const_iterator if elements in c are const

the default constructor initialize an empty container of the specified type except array

create a copy of a container to initialize, container types and element types must match, while use iterators to initialize just need element types can be converted
